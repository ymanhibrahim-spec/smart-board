# حصتي AI - منصة التعلم الذكي المتكاملة 🎓✨
## Hissati AI - All-in-One Smart Classroom Platform

[![Firebase](https://img.shields.io/badge/Firebase-039BE5?style=for-the-badge&logo=Firebase&logoColor=white)](https://firebase.google.com/)

**حصتي AI** هي منصة تعليمية متطورة مخصصة للمعلمين لإدارة الفصول الدراسية الذكية. تجمع المنصة بين أدوات الشرح الرقمية وأنظمة التفاعل المباشر مع الطلاب باستخدام الذكاء الاصطناعي.

---

### 🌟 المميزات الرئيسية (Key Features)

#### 👨‍🏫 للمعلم (Teacher Features)
*   **Smart Whiteboard:** سبورة رقمية احترافية تدعم اللمس، الأشكال الهندسية، والخطوط المتعددة.
*   **Wireless Teaching:** تحويل الهاتف إلى مايكروفون لاسلكي يبث النصوص مباشرة على السبورة.
*   **Formal Quiz Builder:** إنشاء امتحانات متنوعة (اختيارات، صح وخطأ، مقالي، علل) مع نظام درجات آلي.
*   **Anti-Cheat System:** نظام حماية يمنع الطلاب من مغادرة صفحة الامتحان مع إمكانية "العفو" عن الطالب برمجياً.
*   **Live Analytics:** لوحة مراقبة لحظية لنتائج الطلاب وتحليل مستواهم.

#### 🎓 للطالب (Student Features)
*   **Easy Entry:** الدخول عبر QR Code أو كود رقمي بسيط مكون من 6 أرقام.
*   **Interactive Exams:** واجهة امتحانات سلسة تدعم جميع أنواع الأسئلة مع ظهور النتيجة فورياً.
*   **Leaderboard:** لوحة شرف تظهر أفضل الطلاب في الفصل لتحفيز التنافس.

---

### 🛠 التقنيات المستخدمة (Tech Stack)
*   **Frontend:** React.js, Vite, Framer Motion (Animations), Lucide React (Icons).
*   **Backend:** Firebase (Authentication, Firestore Database).
*   **Tools:** html5-qrcode, jsPDF, html2canvas.

---

### 🚀 كيفية التشغيل (Getting Started)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/smart-hissati-board.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Firebase:**
   Create a project on Firebase and add your config to `src/utils/firebase.js`.

4. **Run the app:**
   ```bash
   npm run dev
   ```
